Strava Run krijgt ondersteuning voor M7 processor in iPhone 5s

26 september om 10:50

De nieuwste versie van hardloopapp Strava Run brengt ondersteuning voor de M7 processor in de iPhone 5s met zich mee. De M7 motion coprocessor is een kleine chip in de iPhone 5s, die het voor apps mogelijk maakt om uitgebreide bewegingsdata te gebruiken zonder dat de accu van je iPhone snel wordt leeggetrokken. Doordat Strava Run nu gebruikt maakt van de M7 kan de app nauwkeuriger snelheden en routes bijhouden, terwijl de app minder batterij verbruikt. Ook handig: als je stopt met bewegen pauzeert de app op een iPhone 5s nu automatisch, zonder dat je hier zelf een knop voor hoeft in te drukken.
